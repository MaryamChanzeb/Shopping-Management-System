# Shopping Management System

## Project Overview

The Shopping Management System is a console-based C++ application developed as an academic project.

It allows users to manage products, maintain inventory, process customer orders, and generate checkout receipts.

---

## Features

- Add Product
- Remove Product
- Search Product
- View Stock
- Add Stock
- Add Items to Cart
- Remove Items from Cart
- Checkout System
- Discount Calculation
- File Handling using products.txt

---

## Technologies Used

- C++
- Object-Oriented Programming Concepts
- File Handling
- Arrays
- Functions

## How to Run

1. Download the repository.
2. Open the project in CodeBlocks or Visual Studio.
3. Ensure products.txt is in the project directory.
4. Compile the source file.
5. Run the application.
