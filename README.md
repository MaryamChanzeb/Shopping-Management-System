# 🛒 Shopping Management System

> A simple yet efficient **Console-Based Shopping Management System** developed in **C++** as a **Computer Programming Semester Project**.

---

## 📖 Project Overview

The **Shopping Management System** is a menu-driven console application that helps manage products, inventory, shopping carts, and checkout operations.

The system stores product information in a text file and allows users to perform various shopping and inventory management tasks through an interactive console interface.


## ✨ Features

✅ Add New Product

✅ Remove Product

✅ Search Product

✅ View Available Stock

✅ Add Stock

✅ Add Products to Cart

✅ Remove Products from Cart

✅ Generate Checkout Receipt

✅ Calculate Total Bill

✅ Apply Discounts

✅ Store Product Data using File Handling
